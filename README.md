### INSTALATIONS

### Clone repo : 
```bash
https://github.com/IwanProject/blogspot.git
```
### Install vendor : 
```bash
composer install
```
### Generate key : 
```bash
php artisan key:generate
```

### Migrate database : 
```bash
php artisan migrate
```
### Generate Data Dummy with Seeder
```bash
php artisan db:seed
```


### Run applications
```bash
php artisan serve
```
